## Getting Started

### What is fsWallet?

fsWallet is a decentralized wallet built using Web5 technology, allowing users to send and receive payments without traditional banking intermediaries. It leverages DIDs to ensure privacy, control, and security of user data.

---

## Features

- **Decentralized Identity (DID):** Users can create, manage, and verify their identities securely with no central authority.
- **Cross-Border Payments:** Send and receive payments seamlessly across borders without excessive fees.
- **Secure Transaction History:** Track and manage all transactions securely using Web5 protocols.
- **Secure Credentials:** Store verifiable credentials for financial transactions and identity verification.
