## Setup Instructions

### 1. Create a New Wallet

To begin using Fiatsend, create a wallet using the following steps:

1. Click on **Create Wallet**.
2. Secure your wallet with a passphrase.
3. Backup your decentralized identity (DID) securely.

### 2. Import an Existing Wallet

If you already have a wallet, you can import it by following these steps:

1. Click on **Import Wallet**.
2. Enter your existing DID and private key.
3. Access your funds securely within Fiatsend.
