# Security

## Private Key Management

Fiatsend utilizes secure encryption algorithms to protect user keys. For production environments, private keys are securely managed through **AWS KMS (Key Management Service)**, ensuring that sensitive data never resides on local devices.

## DID Recovery

If you lose access to your wallet, Fiatsend allows **DID recovery** using a backup passphrase. Ensure your backup is stored securely to prevent loss of access.
